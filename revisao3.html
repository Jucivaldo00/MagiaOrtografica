Regras da Fase 3:

  • O objetivo do jogo é encontrar pares de palavras que tenham sons iguais ou escritas iguais, mas significados diferentes (homônimos).
  • As cartas começam viradas para baixo.
  • A cada turno, o jogador pode virar duas cartas de uma vez.
  • Se as palavras formarem um par, o jogador fica com as cartas e joga novamente.
  • Se não formarem um par, as cartas são viradas de volta.
  • O jogador tem 1 minuto para completar o jogo, ou seja, precisa encontrar todos os pares dentro desse tempo para vencer.
  • O jogo é dividido em 3 rodadas. Cada rodada consiste em tentar encontrar os pares de palavras durante o tempo disponível. Se o jogador encontrar todos os pares corretamente dentro das 3 rodadas, ele avança para a Fase 4.

Dicas para o Jogo de Cartas

Aqui estão algumas dicas para ajudá-lo a se preparar:

  • Preste atenção nos sons das palavras. Muitos pares têm som semelhante, mas significados diferentes.
  • Mantenha as cartas na memória e tente lembrar onde os pares estão localizados.
  • Os pares podem ter a mesma escrita ou sons parecidos, então observe bem as palavras.
  • O tempo é limitado! Fique atento ao 1 minuto para encontrar todos os pares.
